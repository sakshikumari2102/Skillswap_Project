**SkillSwap Project**

Developed a multi-page web application to enable peer-to-peer skill exchange and collaboration. The platform allows users to create profiles, explore available skills, connect with others, and communicate through an integrated messaging system. Designed with a modular front-end architecture, the project ensures smooth navigation and an organized user experience.

**Key Features:**

* Built multiple interactive pages including dashboard, profile, skill matching, messaging, and feedback system
* Implemented user authentication and session handling using JavaScript
* Designed a skill-matching interface to connect users based on shared interests
* Developed a messaging module for real-time user interaction
* Created a responsive UI using HTML, CSS, and modular JavaScript files

**Tech Stack:**
HTML, CSS, JavaScript
